First Prize – VISAI 2025 (International Level)

EV Health Monitoring System

An integrated real-time health monitoring and diagnostics suite for Electric Vehicles. Developed to solve a real-world electric vehicle diagnostics problem statement proposed by Mahindra & Mahindra Ltd., securing 1st place at VISAI 2025 hosted by Vel Tech, Chennai.

Key Specifications

  • Collects EV battery temperature, state-of-charge, and vibration parameters using hardware sensors.
  • Employs machine learning classifiers for real-time anomaly detection and predictive wear diagnostics.
  • Includes an augmented reality (AR) dashboard module for technicians to perform quick visual maintenance.
Raspberry Pi IoT Sensors Machine Learning Augmented Reality Python
Second Prize – Malaysia Project Expo (MAHSA University)
Machine Learning & Data Analytics

Smart Energy Forecasting System

A predictive energy forecasting model designed to predict power grid load consumption and optimize residential energy management. Awarded 2nd prize at the MAHSA University Project Expo in Malaysia.

Key Specifications

  • Performs time-series load forecasting using historical consumption grids.
  • Processes weather forecasts, day types, and industrial loads to increase forecasting accuracy.
  • Saves energy consumption costs by scheduling peak-shaving operations automatically.
Machine Learning Time Series Data Analysis Python Predictive Analytics
Second Runner-Up – National Hackathon 2024
Machine Learning & Software

Medicine Recommendation System

An intelligent diagnostic aid that recommends suitable medications based on symptoms using classification techniques. Secured Second Runner-Up and a ₹20,000 cash prize in a 24-hour national hackathon at Kalasalingam University.

Key Specifications

  • Translates unstructured patient symptom entries using text processing.
  • Predicts diseases using multi-label classification algorithms.
  • Cross-references a database to flag high-risk drug combinations and alert users.
Machine Learning NLP Python Classification Models Healthcare Tech
Third Prize – VISAI 2026 (International Level)
Industrial IoT & ML

Machine Health Monitoring System

An industrial predictive maintenance system that detects vibrations and temperature fluctuations to forecast breakdown. Secured 3rd place at the VISAI 2026 International Project Competition.

Key Specifications

  • Attaches accelerometer sensors to measure high-frequency mechanical vibration.
  • Runs edge-computing anomaly classification to prevent sensor-to-cloud latency.
  • Generates automatic scheduled work orders for parts replacement prior to expected failure.
